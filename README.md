# tetsubin-db

南部鉄瓶（盛岡・水沢）の工房と銘柄を横断検索できるデータベースサイト。

- **本番**: https://tetsubin-db.com
- **プラットフォーム**: 静的HTML + Cloudflare Pages（main への push で自動デプロイ）
- **デザイン**: 黒（#0a0a0a）×銅（#b87333）×金（#d4af37）、SVGアイコン統一

## サイト構造

- 模様 / 容量 / 熱源 / 予算 の4軸逆引きデータベース
- 本物 vs 中国製偽物 見分け方ガイド
- 工房詳細（盛岡系 / 水沢系、一次ソース検証済み）
- ふるさと納税（奥州市・盛岡市）

## 編集方針

掲載情報はすべて**公式サイト（一次ソース）で確認したもののみ**。架空の銘柄・推測価格・裏付けのない受賞歴は掲載しない（嘘ゼロ）。
